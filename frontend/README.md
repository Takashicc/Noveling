# Frontend

## Environment

- Package Manager
  - yarn
- Language
  - TypeScript
- Framework
  - Next.js
- UI
  - Mantine
- Lint & Format
  - TypeScript ESLint
  - Rome
- Others
  - Storybook

## Set up

- Install dependencies

  ```sh
  yarn install
  ```

- Run dev environment

  ```sh
  yarn dev
  ```
